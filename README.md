## :construction_worker: How to run
```bash
# Clone Repository
$ git clone https://github.com/Victor19Rodrigues/linkedin-ui-clone.git
```

### üíª Run Web Project

```bash
# Install dependencies
$ yarn install

# Run aplication
$ yarn start
```
Go to http://localhost:3000/ to see the result.
